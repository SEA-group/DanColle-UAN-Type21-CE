## Installation
* **For WGC users**, copy `content` and `particles` into `[WoWS folder]/bin/[largest number]/res_mods/`
* **For Steam users**, copy `content` and `particles` into `/steamapps/common/World of Warships/bin/[largest number]/res_mods/`
